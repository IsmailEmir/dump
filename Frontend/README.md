# SaveYourTime

Веб-приложение **SaveYourTime** — менеджер задач с поддержкой команд, профиля пользователя и локализации. Это фронтенд-часть проекта: одностраничное приложение на React, которое общается с REST API бэкенда.

## Возможности

- **Авторизация** — вход и регистрация (`/login`).
- **Задачи** — создание, редактирование, удаление, карточки задач, выбор даты и времени.
- **Команды** — создание команд, участники, задачи в контексте команды, передача руководства и другие сценарии работы с группой.
- **Профиль и настройки** — модальные окна профиля, смена пароля, выход из аккаунта.
- **Интерфейс** — светлая и тёмная тема (сохраняются в `localStorage`), прелоадер при загрузке основного экрана.
- **Локализация** — переключение языка через контекст `LanguageProvider` и файлы переводов.

## Технологии

- [React](https://react.dev/) 19
- [React Router](https://reactrouter.com/) 7
- [Axios](https://axios-http.com/) — HTTP-клиент к API
- [date-fns](https://date-fns.org/), [react-datepicker](https://github.com/Hacker0x01/react-datepicker) — работа с датами в задачах
- [Testing Library](https://testing-library.com/react) — тесты компонентов

## Требования

- [Node.js](https://nodejs.org/) (LTS рекомендуется)
- Запущенный бэкенд с API (по умолчанию ожидается адрес из `src/services/api.js`)

## Установка и запуск

```bash
npm install
npm start
```

Приложение откроется в браузере (обычно `http://localhost:3000`).

## API

Клиент настроен на базовый URL **`http://localhost:5257/api`** (см. `src/services/api.js`). Для локальной разработки бэкенд должен быть доступен по этому адресу или URL нужно изменить в конфигурации API.
