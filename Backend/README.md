# SaveYourTime — backend API

ASP.NET Core Web API приложениe для управления задачами: личные задания пользователя, команды, роли и командные задачи (`TeamAssignment`).

## Возможности

- Регистрация и вход; сессия на **cookie** (`AuthToken`, HttpOnly).
- CRUD пользователей, ролей, команд, назначений (assignments) и командных заданий.
- Личные задачи с **статусом**, **приоритетом** и **дедлайном**.
- Связь пользователей с командами (many-to-many), лидер команды, загрузка **аватарки команды** в `wwwroot/uploads`.
- Описанные методы в Swagger
- Обработка исключений через middleware, раздача статических файлов.

## Стек

| Компонент        | Версия / продукт     |
|-----------------|----------------------|
| Runtime         | .NET 10              |
| ORM             | Entity Framework Core 10|
| БД              | PostgreSQL |
| Документация API| Swagger  |
| Аутентификация  | Cookie |

## Структура проекта Clean Architecture

- **Domain** — хранит модели для ORM и интрефейсы для сервисов и репозиториев.
- **Application** — контроллеры, DTO, реализации сервисов.
- **Infrastructure** — Внутренние сервисы, работа с ORM.

## Требования для запуска проекта

- [.NET 10 SDK](https://dotnet.microsoft.com/download)
- Сервер PostgreSQL и строка подключения

## Быстрый старт

1. Клонируйте репозиторий и перейдите в каталог с проектом:

   ```bash
   cd SaveYourTime
   ```

2. Настройте строку подключения в файле appsetings.js

3. Примените миграции (из каталога `SaveYourTime`):

   ```bash
   dotnet ef database update
   ```

4. Запуск:

   ```bash
   dotnet run
   ```

CORS policy `AllowReactApp`: методы и заголовки разрешены, credentials включены — фронт на `localhost:3000` может отправлять cookie.

## Связка с фронтендом

Убедитесь, что React-приложение ходит на тот же origin/порт, что и API, или настроены прокси/CORS так, чтобы запросы с `credentials: 'include'` доходили до API с разрешённого origin.
